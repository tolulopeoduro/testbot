# Groq Chatbot

A modern React chatbot powered by Groq's API using Vite and Tailwind CSS.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Add your Groq API key:
   - Get your key from [console.groq.com](https://console.groq.com)
   - Update `GROQ_API_KEY` in `src/components/Chatbot.jsx`

3. Run the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

## Features

- Real-time chat with Groq's Llama 3.3 70B model
- Responsive UI with Tailwind CSS
- Smooth scrolling to latest messages
- Loading indicators
- Message history display
